# Summary

> 만들고 싶어서 만들어본 Spring boot를 사용한 웹 사이트
화면의 왼쪽에 제시되는 긴 길이의 지문을 적당히 요약하면, AI모델이 사용자가 지문을 얼마나 잘 요약했는지 평가하여 그 결과를 사용자에게 표시하는 웹 사이트.

## 기술 스택
- 언어 : `Java`
- 프레임워크 : `Spring`
- 사용 DB : `MySQL(Azure)`
